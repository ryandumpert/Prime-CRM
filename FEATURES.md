## 🏆 Tier 1 — High-Impact Quick Wins

| # | Feature | Status | Completed |
|---|---------|--------|-----------|
| 1 | **Automated Follow-Up Reminders & Snooze** — Prompt advisors to set next follow-up date when logging interactions. Snooze buttons on cards (1d, 3d, 1w). | ✅ Done | Mar 2, 2026 |
| 2 | **Lead Scoring** — Score 0–100 based on recency, interactions, priority, entry date, contact completeness. Displayed as badge on cards, sorted by score. | ✅ Done | Mar 2, 2026 |
| 3 | **Quick-Action Buttons on Pipeline Cards** — Tap-to-call, tap-to-text (mobile only), tap-to-email on cards. Reduces friction to reach leads. | ✅ Done | Mar 2, 2026 |
| 4 | **Activity Timeline Filters** — Filter pills on lead detail activity tab: All, Calls, Texts, Emails, Notes, Status Changes. Each shows count badge. | ✅ Done | Mar 2, 2026 |
| 5 | **Bulk Actions: Reassign, Status, Priority** — Multi-select leads → bulk reassign to advisor, change status, change priority, or archive. | ✅ Done | Mar 2, 2026 |
